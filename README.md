## Usage

### インポート方法
```
from mewcalc import MewCalc
```

### 猫の年齢を生まれたdateから計算する
```
from datetime import datetime, date, time
mc = MewCalc()
born = date(1998, 1, 4)
res = mc.calculate(born)
```

### 注意

2016年以降は無理です。
